## 📌 About the Project

A collection of advanced performance analysis and telemetry tools for Call of Duty Black Ops 7. Each tool provides real-time tracking, visual overlays, and gameplay improvement features.

---

## 🔧 Available Tools

- **FECURITY** — Performance analyzer with customizable aimbot, player and loot tracking, and streamproof overlays for competitive play.
- **FROST** — Ultra-responsive analyzer with customizable aimbot, advanced ESP, and sophisticated anti-detection for high-rank play.
- **ANCIENT** — Humanized aimbot analyzer with comprehensive ESP, triggerbot, and no-recoil for fast-paced multiplayer and Warzone.
